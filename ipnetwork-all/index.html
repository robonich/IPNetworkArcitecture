2015年度 後期学生実験「IPネットワークアーキテクチャ」

東京大学電子情報工学科 2015年度 後期学生実験 江崎・落合研担当の「IPネットワークアーキテクチャ」の講義用 Web サイトです。

連絡事項

  • 場所: 243号講義室
  • 実験には学科で貸与されたPCを持参してください。OSはUbuntuを利用します。
  • 実験中はネットワークマネージャを切るためインターネットが利用できません。以下の講義資料のダウンロード、およびパッケージのインストールは実験前にしておいてください。

実験で利用するパッケージ


$ sudo apt-get update

($ sudo apt-get upgrade)

$ sudo apt-get install tcpdump wireshark traceroute quagga bind9 dnstracer apache2 cu

シリアルコンソールの使い方


$ sudo chgrp root /dev/ttyUSB0

$ sudo cu -l /dev/ttyUSB0 -s 115200
